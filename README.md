# Build

Windows

    > git clone https://github.com/seven332/Nimingban
    > cd Nimingban
    > git submodule update --init
    > gradlew app:assembleDebug

Linux

    $ git clone https://github.com/seven332/Nimingban
    $ cd Nimingban
    $ git submodule update --init
    $ ./gradlew app:assembleDebug

生成的 apk 文件在 app\build\outputs\apk 目录下

The apk is in app\build\outputs\apk
